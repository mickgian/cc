# RapidApi Key
In order for the App to work you need to insert your RapidApi key into
the AuthorizationInterceptor.kt file.
You'll find this:

    const val RAPID_API_KEY_VALUE = "Insert Your Key Here"

Put your RapidApi key in place of that.
